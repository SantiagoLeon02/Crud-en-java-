CRUD DE PRODUCTOS CON JAVA USANDO ARRAYLIST (CONSOLA)

Este es un programa de consola en el cual permite gestionar una lista de productos basicos

Funcionalidades 

1. Agregar productos
2. Mostrar la lista de productos
3. Modificar el producto
4. Eliminar un producto individualmente o vaciar toda la lista 

Tecnologias usadas 

- Java 8+
- ArrayList
- Scanner
- Programación estructurada

Cómo ejecutarlo:

1. Clona el repositorio o descarga el código.
2. Abre el archivo `Main.java` en tu IDE (IntelliJ, Eclipse, VS Code, etc.)
3. Ejecuta el método `main()` para iniciar el programa.
